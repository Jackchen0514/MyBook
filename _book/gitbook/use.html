Gitbook基本使用

GitBook基本使用

GitBook初始化

gitbook --version

执行 gitbook --version 错误：Error: Cannot find module 'config-chain'

cannot find module 模块, 安装该模块即可

npm install -g config-chain

创建项目

mkdir book
cd book
gitbook init

创建好的项目目录包含一下文件

ovirt-branding.md  README.md  SUMMARY.md
