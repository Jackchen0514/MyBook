Gitbook

简介

GitBook 是一款基于 Node.js 开发的开源的工具，可以通过命令行的方式创建电子书项目，再使用 MarkDown 编写电子书内容，然后生成 PDF、ePub、mobi 格式的电子书或生成一个静态站点。
