# Fiddler抓包

## 安装

首先，安装Mono环境，在Ubuntu环境下安装很简单，输入：
```
sudo apt-get install mono-complete
```

## 下载Fiddler

[下载地址](http://ericlawrence.com/dl/MonoFiddler-v4484.zip)

## 启动Fiddler

```
mono Fiddler.exe
```
